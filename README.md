# 🌍 Tourism Management System (Java Full Stack)

A full-stack Tourism Management System developed using Java technologies. This application allows users to browse travel destinations, view tour packages, register/login, and book trips through an easy-to-use interface.

## 🚀 Features

- User Registration & Login
- Browse Tour Packages
- View Destination Details
- Book Travel Packages
- Admin Dashboard
- Manage Tour Packages
- Manage Users
- Responsive User Interface
- Secure Authentication
- Database Integration

## 🛠️ Technologies Used

### Frontend

### Backend

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/yourusername/tourism-java-fullstack.git
```

### Navigate to the Project

```bash
cd tourism-java-fullstack
```

### Configure Database

- Install MySQL.
- Create a database named `tourism`.
- Import the SQL file.

### Run the Backend

- Open the project in Eclipse or IntelliJ IDEA.
- Build the project using Maven.
- Run the application.

### Run the Frontend

Open the frontend in your browser or start the frontend server if applicable.
